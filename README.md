# MyFirstGame2D
A 2D platformer game developed for learning purposes using Unity. This project was built to practice and understand core game development concepts.

This is my first Unity project. The goal was not to make a polished game, but to learn and apply fundamental game development mechanics from scratch.

What I Learned & Implemented

**Player Mechanics**
- Character movement, jumping, and attacking 
- Player death and respawn logice etc.

**Enemy System**
- Basic enemy AI with movement, attack, and death etc..

**UI & Canvas**
- Health bar system
- Level transition system etc.

**Audio**
- Background music integration
- Sound management

**Settings**
- In-game settings menu

## 💻 Built With
- Unity
- C#
- Platform: PC (2D)
